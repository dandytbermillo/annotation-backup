Camera POC Test Page

This page validates the camera-based positioning system for the unified canvas nodes implementation. The POC demonstrates camera-based panning, node dragging with proper coordinate transformation, zoom functionality, and edge auto-scroll behavior.

Test Scenarios In Progress

1. Camera Pan

  • Drag empty canvas area to pan
  • Camera X/Y updates in real-time
  • All nodes move together
  • Pan is smooth at all zoom levels

2. Node Dragging

  • Drag individual nodes to reposition
  • Node lands under cursor on release
  • Works correctly at zoom ≠ 1
  • No position jumps

3. Zoom Behavior

  • Scroll wheel zooms in/out
  • Zoom range: 50% - 200%
  • Positions scale correctly
  • Pan/drag still accurate after zoom

4. Edge Auto-scroll

  • Drag node to viewport edge
  • Camera auto-scrolls smoothly
  • Works at all zoom levels
  • Node position updates correctly

Math Validation

Key formulas being tested:

  • Screen to World: worldDelta = screenDelta / zoom
  • Camera Pan: camera.x += dxScreen / zoom, camera.y += dyScreen / zoom
  • Node Position: screenPos = worldPos * zoom + camera
  • Transform: translate(camera.x, camera.y) scale(zoom)

Expected Results

  • ✓ Camera pan moves all content together without individual DOM manipulation
  • ✓ Node dragging is accurate at all zoom levels
  • ✓ Edge auto-scroll integrates smoothly with camera system
  • ✓ Performance is acceptable with multiple nodes
  • ✓ No desync between visual position and stored coordinates
