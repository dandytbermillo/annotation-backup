# Demo Widget

## Overview
Demo Widget is a sample widget used for testing chat integration and list
previews.

## Where it appears
- Dashboard as the Demo Widget (if installed).
- Drawer panel when opened.

## Key behaviors
- "Open Demo Widget" opens the drawer panel.
- Provides a simple list preview in chat.
