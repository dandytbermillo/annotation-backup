# Floating Toolbar

## Overview
The floating toolbar is a context-aware navigation bar that appears on the
canvas. It provides access to recents, organization, tools, and branch actions.

## Where it appears
- On the canvas workspace when working with notes.
- Floating position adjusts based on canvas state.

## Key sections
- Recents: Shows recently accessed notes for quick switching.
- Organization: Opens the folder/knowledge base tree view.
- Tools: Additional canvas utilities.
- Branches: Manage annotation branches on the current panel.
- Format: Text formatting options when editing.
- Add Component: Add sticky notes, timers, or other components.

## Key behaviors
- Click a section icon to expand its panel.
- Hover over items for previews.
- Create new notes directly from recents.
- Navigate folders and subfolders from organization.
