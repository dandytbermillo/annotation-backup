# Entry

## Overview
An entry is a top-level container. Each entry has its own dashboard and one or
more workspaces.

## Where it appears
- The dashboard list of entries.
- Quick Links can point to entries or workspaces.

## Key behaviors
- Entry dashboard shows widgets and panels for that entry.
- Workspaces inside an entry hold notes.
