# Links Panel

## Overview
Links Panel is a text editor widget where you can write notes and highlight text
to create workspace links. Multiple Links Panels can exist with badges (A, B, C, D).

## Where it appears
- Dashboard as Links Panel widgets.
- Drawer panel when opened.

## Key behaviors
- Select any text to show a "Link to Workspace" toolbar.
- Created links navigate to the target workspace when clicked.
- Each panel has a badge letter (e.g., Links Panel D).
- Supports both internal workspace links and external links.
- Deleted links can be restored from panel trash.
