# Recent

## Overview
Recent shows your most recently opened items for the current entry.

## Where it appears
- Dashboard as the Recent widget.
- Drawer panel when opened.

## Key behaviors
- Shows a list of recently opened workspaces and entry dashboards.
- "Show recent" opens the Recent panel in the drawer.
- "List recent in chat" shows a short preview list in chat.
