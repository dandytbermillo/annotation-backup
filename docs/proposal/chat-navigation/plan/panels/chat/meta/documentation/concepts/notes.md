# Notes

## Overview
Notes are the primary content inside a workspace. Multiple notes can be open at
once, with one active note.

## Where it appears
- Inside a workspace view.
- In the Open Notes dock list.

## Key behaviors
- Notes open in tabs/cards on the workspace canvas.
- "Which notes are open?" reports the open notes list.

## Example questions
- "Open note Project Plan"
- "Which notes are open?"
- "Explain notes"
