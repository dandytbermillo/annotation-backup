# Category Navigator

## Overview
Category Navigator is a widget that shows all workspace links from all Quick
Links panels in the current dashboard. It provides a unified view of linked entries.

## Where it appears
- Dashboard as a Category Navigator widget.
- Drawer panel when opened.

## Key behaviors
- Aggregates links from all Quick Links panels.
- Expandable sections for each Quick Links panel.
- Search to filter links across all panels.
- Preview popup shows panel content on hover.
- Trash section to restore deleted panels.
