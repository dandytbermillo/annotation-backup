# Note Actions

## Overview
Notes live inside workspaces. Note actions depend on the current workspace context.

## Supported actions
- Open a note by title
- Answer questions about open notes (when in a workspace)

## Examples
- "open note Project Plan"
- "which notes are open?"
- "what note is open?"

## Behavior notes
- If you ask about open notes while on the dashboard, the chat will ask you to open a workspace first.
- If a note title is not found, the chat responds with a clear "not found" message.
