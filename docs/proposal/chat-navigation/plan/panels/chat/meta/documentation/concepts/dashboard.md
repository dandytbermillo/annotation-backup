# Dashboard

## Overview
The dashboard is the widget grid view inside an entry. It shows widgets and
panels that provide quick access to content.

## Where it appears
- Entry dashboards (including Home).
- Visible when you are not inside a workspace.

## Key behaviors
- Widgets are visible on the dashboard.
- Double-clicking a widget opens its full panel in the drawer.
- "Go to dashboard" returns you to the entry dashboard.
