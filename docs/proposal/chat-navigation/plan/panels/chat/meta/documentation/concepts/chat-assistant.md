# Chat Assistant

## Overview
The chat assistant is an AI-powered interface for navigating and learning
about the app. Ask questions in natural language to get help, navigate to
locations, or understand features.

## Where it appears
- Right side of the screen as a slide-out panel.
- Available from any view (dashboard or workspace).

## Key behaviors
- Ask "what is X?" to learn about features.
- Say "go to X" or "open X" to navigate.
- Ask "search my notes for X" to find content.
- Pills appear for disambiguation when multiple options match.
- Suggestions appear based on context.

## Example questions
- "Go to dashboard"
- "Open workspace Sprint 6"
- "What is a widget?"
- "Search my notes for meeting"
- "Show quick links D"

## Related concepts
Dashboard, Workspace, Widgets, Panels
