# Widgets

## Overview
Widgets are modular components shown on the dashboard. Each widget can open a
full panel in the drawer for more interaction.

## Where it appears
- Entry dashboards as a grid of widgets.
- Some widgets also have panels in the drawer.

## Key behaviors
- Visible widgets can be asked about in chat.
- Double-clicking a widget opens its drawer panel.

## Example questions
- "What widgets are visible?"
- "Open Widget Manager"
- "Explain widgets"

## Related concepts
Dashboard, Panels
