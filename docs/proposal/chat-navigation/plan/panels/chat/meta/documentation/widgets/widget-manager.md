# Widget Manager

## Overview
Widget Manager lets you install, enable, disable, and remove widgets.

## Where it appears
- Dashboard as a widget.
- Drawer panel when opened.

## Key behaviors
- "Open Widget Manager" opens the drawer panel.
- Shows installed widgets and their chat commands.
