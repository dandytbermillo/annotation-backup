# Workspace

## Overview
A workspace is where notes live. Each workspace belongs to an entry and has its
own open notes and active note.

## Where it appears
- Inside an entry when you open a workspace.
- Workspaces can be opened from the dashboard or chat.

## Key behaviors
- Notes are opened and closed within a workspace.
- The dock shows the current open notes.
- "Go to dashboard" exits the workspace view.

## Example questions
- "Open workspace Sprint 6"
- "Which notes are open?"
- "Explain workspace"
