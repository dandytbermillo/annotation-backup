# Panels

## Overview
Panels are the full drawer views of widgets. They open on the right side and
contain the full interactive UI for a widget.

## Where it appears
- Drawer on the right side of the dashboard.

## Key behaviors
- Only one panel drawer is open at a time.
- "What panel is open?" returns the current drawer title.
