# Quick Links

## Overview
Quick Links is a widget that stores shortcuts to workspaces or entries. Multiple
Quick Links panels can exist with badges (A, B, C, D, E).

## Where it appears
- Dashboard as Quick Links widgets.
- Drawer panel when opened.

## Key behaviors
- Each panel has a badge letter (e.g., Quick Links D).
- "Show quick links" may ask which panel if multiple exist.
- "List quick links D in chat" shows a preview list.

## Example questions
- "Show quick links"
- "Open quick links D"
- "List quick links D in chat"

## Related concepts
Dashboard, Panels
