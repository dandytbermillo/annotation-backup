# Notes

Notes are the core content units in the system. Each note contains rich text that you can edit, format, and annotate. Notes live inside workspaces and can be organized in the Knowledge Base.

## Creating Notes

- Open a workspace
- Use the "New Note" button or command
- Start typing in the editor

## Note Features

- **Rich Text**: Format with headings, lists, links, and more
- **Auto-save**: Changes saved automatically
- **Branching**: Create annotation branches from selections
- **Organization**: Organize notes in folders within Knowledge Base

## Finding Notes

- Use the Navigator panel to browse
- Search by title or content
- Check Recent for recently accessed notes
