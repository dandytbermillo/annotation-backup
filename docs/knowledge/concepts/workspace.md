# Workspace

A workspace is where your notes live. You can create, edit, and organize notes within a workspace. Each workspace provides a focused environment for working with related content.

## What You Can Do

- Create and edit notes
- Open multiple notes at once
- Navigate between notes using the Open Notes dock
- Use the canvas to arrange your notes spatially

## Key Features

- **Note Editor**: Rich text editing with TipTap
- **Open Notes Dock**: See and switch between open notes
- **Canvas View**: Arrange notes on an infinite canvas
- **Auto-save**: Your changes are saved automatically
