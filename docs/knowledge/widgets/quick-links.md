# Quick Links Widget

Quick Links provides shortcuts to your bookmarked or frequently used items. You can have multiple Quick Links panels (A, B, C, D, E) for different categories.

## Features

- Bookmark important notes and workspaces
- Organize links by category
- Quick access from the dashboard
- Multiple panels for different link groups

## How to Use

- Click a link to navigate directly
- Use chat: "open quick links" or "open quick links D"
- Add new links through the panel's add button
