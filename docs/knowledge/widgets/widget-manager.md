# Widget Manager

The Widget Manager lets you customize your dashboard by adding, removing, and configuring widgets. You can install new widgets or manage existing ones.

## Features

- View all available widgets
- Install new widgets
- Remove widgets you don't need
- Configure widget settings

## How to Use

- Use chat: "open widget manager"
- Browse available widgets
- Click to install or configure
