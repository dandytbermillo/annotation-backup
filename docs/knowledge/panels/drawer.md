# Drawer Panel

The drawer is a side panel that opens when you interact with widgets or use certain commands. It shows detailed content without leaving the dashboard.

## How It Works

- Opens from the right side of the screen
- Shows widget details or expanded content
- Can be closed by clicking outside or pressing Escape

## Common Uses

- Viewing Quick Links details
- Browsing Recent items
- Navigating folder structure
- Managing settings
