Debugging Cmd+B / Ctrl+B Shortcut

Issue Analysis:

The Cmd+B (Mac) or Ctrl+B (Windows/Linux) shortcut is not working. Possible reasons:

  1. Keyboard event is being intercepted by another handler
  2. The sidebar toggle function is not properly connected

How to Enable the Feature:

The multi-layer canvas is now always enabled; no console tweaks are required.
